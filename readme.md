# GitHub Profile Readme Stats (Forked for Personal Use)

This repository is a **personal fork** of [GitHub Readme Stats](https://github.com/anuraghazra/github-readme-stats). It is **self-hosted on Vercel** and used to generate GitHub stats cards for my personal GitHub profile README.

## Live Deployment

My self-hosted version is available here:  
[https://github-profile-readme-stats.vercel.app](https://github-profile-readme-stats.vercel.app)

## Usage Notes

- The repository is connected to **Vercel**, which auto-deploys updates when commits are pushed. 
- This deployment uses a personal GitHub token to avoid API rate limits.   
- API endpoints can be used in a GitHub profile README like this:

```md
[![Top Languages](https://github-profile-readme-stats.vercel.app/api/top-langs/?username=zied-bouazizi&layout=compact)](https://github.com/zied-bouazizi/github-profile-readme-stats)
